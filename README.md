
 How to Install and run on android
------------
`npm install -g react-native-cli `\
`cd react-native-quick-sample`\
`npm install`\
`react-native run-android`

How to  Install and run on ios
--------

`npm install -g react-native-cli`\
`cd react-native-quick-sample`\
`npm install`\
`react-native run-ios`
